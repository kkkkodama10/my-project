# テーブル・カラム 英日対応表

## pets テーブル（ペット情報）

| 英語名（物理名）     | 日本語名（論理名） | 備考                                               |
| ------------ | --------- | ------------------------------------------------ |
| `id`         | ID        | 自動採番 (SERIAL / BIGINT)                           |
| `name`       | 名前        | 最大 50 文字                                         |
| `species`    | 種別        | 最大 30 文字                                         |
| `sex`        | 性別        | ENUM: `male` = オス, `female` = メス, `unknown` = 不明 |
| `created_at` | 登録日       | タイムスタンプ (作成時)                                    |
| `updated_at` | 更新日       | タイムスタンプ (更新時 自動上書き)                              |

### 性別 ENUM 対応表

| 値 (英語)    | 表示名 (日本語) |
| --------- | --------- |
| `male`    | オス        |
| `female`  | メス        |
| `unknown` | 不明        |
