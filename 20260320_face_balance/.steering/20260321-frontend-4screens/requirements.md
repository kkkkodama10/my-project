# 要求仕様: フロントエンド4画面

## 概要

FaceGraph の React フロントエンドを実装する。
4画面（人物一覧・人物詳細・比較・比較履歴）を React Router v6 でルーティングし、
TanStack Query でサーバー状態を管理する。

## 画面一覧

| 画面 | パス | 主な操作 |
|------|------|---------|
| 人物一覧 | `/` | 人物カード一覧・人物追加・人物削除 |
| 人物詳細 | `/persons/:id` | 画像アップロード・画像削除・解析ステータス表示 |
| 比較 | `/compare` | 2人選択・スコア表示（0〜100%）・キャッシュ判別 |
| 比較履歴 | `/history` | 過去の比較結果一覧（有効/無効ステータス付き） |

## 受け入れ条件

### 人物一覧（`/`）
- [ ] 登録済み人物をカード形式で一覧表示する（名前・画像枚数）
- [ ] 名前入力フォームで人物を追加できる
- [ ] 人物カードの削除ボタンで人物を削除できる（確認なし）
- [ ] カードをクリックして人物詳細へ遷移できる
- [ ] 「比較する」ナビゲーションで `/compare` へ遷移できる

### 人物詳細（`/persons/:id`）
- [ ] 人物名と登録画像一覧を表示する
- [ ] ファイル選択で画像をアップロードできる
- [ ] 各画像のステータス（`uploaded`/`validating`/`analyzing`/`analyzed`/`error`）をバッジで表示する
- [ ] `error` ステータスの場合、`metadata.error` のメッセージを表示する
- [ ] 解析中ステータス（`analyzed` 以外）の場合、ポーリングで自動更新する（3秒間隔）
- [ ] 画像削除ボタンで画像を削除できる
- [ ] 一覧へ戻るリンクを表示する

### 比較（`/compare`）
- [ ] ドロップダウン2つで人物A・人物Bを選択できる
- [ ] 「比較する」ボタンで類似度スコアを表示する
- [ ] スコアは 0〜100% で表示する
- [ ] キャッシュ利用の場合「キャッシュ」バッジを表示する
- [ ] 比較履歴へのリンクを表示する

### 比較履歴（`/history`）
- [ ] 全比較結果を日時降順で一覧表示する
- [ ] 各比較に人物名・スコア・有効/無効ステータスを表示する
- [ ] 比較ページへ戻るリンクを表示する

## 非機能要件

- Tailwind CSS でスタイリング（クラスのみ、カスタムCSS最小限）
- API エラー時はトーストでエラーメッセージ（日本語）を表示する
- ローディング中はスピナーを表示する
- MinIO の画像を表示する際は `/api` プロキシ経由でなく、サムネイル URL を使用する
